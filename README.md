# [Flash](https://flash.now.sh/)

A speed reading web app inspired by [Spdr](https://www.producthunt.com/posts/spdr) that allows you to read large text crazy fast by rapidly showing words one by one.

## Features

* Light/Dark theme
* Custom rate
* Clean interface
